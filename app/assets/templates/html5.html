HTML 5

New Tags

HTML 5 adds a whole bunch of new tags and some new concepts to web development that are pretty exciting. HTML 5 offers a bunch of new markup tags, along with some really cool new tags for implementing new multimedia API's

Multimedia

<video>, <audio>, and <canvas> are just a few of the tags added in that allow easier implementation of multimedia. Unfortunatley, just like everything in life, there are advantages and disadvantages to using these tags. For starters. Not all browsers have implemented these features, so you still must provide a fallback solution if you want to provide the same experience to all users of your application or website. On the other hand, for users that do have a browser supporting these features, you can provide multimedia without resorting to an external library. You no longer need to include a flash plugin to play music or videos, which can significantly increase your page load time, greatly improving your user's experience.

Markup

HTML 5 also adds a plethora of new markup tags, which in most cases replace and or supplement existing tags, providing developers with more specific ways of delivering their content; <section>, <article>, <header> and <nav>. Just like the multimedia tags, these are only useful in browsers that support them, so be careful with your usage. The <header> and <nav> tags are both failry self explanatory, and are often used in conjuction. The <header> tag provides a header for the page, much like the header you see on this page. You can also easily include a nav component within as follows. <section> and <article> tags are a little more complex, and we will go into greater detail on usage in a little bit.

API's

With HTML 5, you get access to a new set of API's that make life a little easier when designing applications. These API's are a little bit trickier to use however, and there are plenty of good tutorials on usage out there, so I am simply going to list them. (Thanks wikipedia!)

  • The canvas element for immediate mode 2D drawing
  • Timed media playback
  • Offline Web Applications
  • Document editing
  • Drag-and-drop
  • Cross-document messaging
  • Browser history management
  • MIME type and protocol handler registration
  • Microdata
  • Web Storage, a key-value pair storage framework that provides behaviour similar to cookies but with larger storage capacity and improved API
  • Geolocation
  • Web SQL Database, a local SQL Database (no longer maintained).
  • The Indexed Database API, an indexed hierarchical key-value store (formerly WebSimpleDB).
  • HTML5 File API handles file uploads and file manipulation.
  • Directories and System, an API intended to satisfy client-side-storage use cases not well served by databases.
  • File Writer, an API for writing to files from web applications.
  • Web Audio API, a high-level JavaScript API for processing and synthesizing audio in web applications.

Sections / Articles

<section> and <article> tags are two of the more interesting tags. Like <header> and <nav> tags, they operate much like more specific <div> tags, but unlike <header> and <nav> tags, they are a little less self explanatory, and using them properly is a little more tricky.
